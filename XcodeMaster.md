# Xcodeショートカットのあれこれ

## 基本操作

### command + R

ビルドとランを同時実行

### command + .

ラン状態の停止

### command + Q

シュミレーターの終了

### command + shift + K

クリーン（間違ってないのにエラーが発生する場合はとりまクリーン）

### command + control + ←

戻るのような動作（ファイルを跨いでる場合）

### command + control + →

進むのような動作

### control + E

行の最後に移動

### control + A

行の先頭に移動
